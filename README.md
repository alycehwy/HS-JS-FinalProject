# HS-JS-FinalProject

床具預購系統 － RESTful API 前後台系統介接

## Description

JavaScript 工程師養成直播班課程最終關卡。
利用六角學院所提供床具預購系統的 UI 設計稿，
介接 11 支 RESTful API，完成預購系統產品列表、購物車、新增訂單、後台訂單管理等功能。
